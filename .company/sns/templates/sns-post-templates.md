# SNS投稿テンプレート集

SNS投稿を作るときに使うプロンプトテンプレート集。体験メモを貼るだけで毎回使える形式でストックしていく。

---

## テンプレート1: 読者の悩みから始まる記事の設計図

**用途**: 自分の体験メモを「自分語り」で終わらせず、読者の悩みを起点にした記事の設計図に変換する。

**使い方**: 下のプロンプトをコピーし、【体験メモ】欄に自分のメモを貼り付けてAIに渡すだけ。

### プロンプト（コピーして使う）

```
あなたは読者心理に強い編集者です。
以下の私の体験メモを「読者の悩みから始まる記事の設計図」に変換してください。

【体験メモ】
（ここに体験メモを貼る）

次の5つの問いに答える形で、下の穴埋めフォーマットをすべて埋めてください。

1. この体験は、誰のどんな悩みにつながるか
   → 読者像: ＿＿＿＿
   → 悩み: ＿＿＿＿

2. 読者はどんな場面でこの話を読みたくなるか
   → 場面: ＿＿＿＿

3. 体験談の前に置く「読者の入口」の一文
   → 入口の一文: ＿＿＿＿

4. 読者が明日そのまま真似できる最小単位
   → 最小アクション: ＿＿＿＿

5. 読み終えた読者に見直してほしいこと
   → 見直しポイント: ＿＿＿＿

条件:
- 体験談の自慢話・報告にせず、必ず読者の悩みを起点にすること
- 「入口の一文」は読者が自分ごと化できる問いかけ・共感の一文にすること
- 「最小アクション」は明日そのまま真似できる、5分以内にできる粒度まで具体化すること
- 「見直しポイント」は読者が自分の行動や状態を振り返れる形にすること
```

### 変換例（1組）

体験メモの例:
「平日は本業で疲れて副業が進まなかったが、朝出勤前に15分だけ作業する形に変えたら、1週間続いて記事が1本書き上がった」

#### ❌ 悪い変換例（書き手起点・自分語り）

- 読者像: みんな
- 悩み: 副業が続かない
- 場面: いつでも
- 入口の一文: 「朝活を始めたら記事が1本書けました！」
- 最小アクション: 「毎日コツコツ頑張りましょう」
- 見直しポイント: 「継続は力なりです」

→ ダメな理由: 読者像が曖昧で誰にも刺さらない。入口が自分の成果報告になっていて読者の悩みから始まっていない。アクションが精神論で、明日そのまま真似できない。

#### ⭕ 良い変換例（読者の悩み起点）

- 読者像: 本業のあとに副業をやろうとして、いつも疲れて手が付かない会社員
- 悩み: 「夜やろう」と思っても気力が残っておらず、副業が何週間も止まっている
- 場面: 夜、今日も副業に手を付けられなかった罪悪感の中でスマホを眺めているとき
- 入口の一文: 「『帰ったらやろう』が今日も守れなかった——悪いのは意志じゃなくて、時間帯かもしれません」
- 最小アクション: 明日の朝、いつもより15分だけ早く起きて、出勤前に副業のファイルを開いて1行だけ書く
- 見直しポイント: 1週間後、「夜にやろうとしていた頃」と比べて手を付けられた日が何日増えたかを数えてみる

→ 良い理由: 読者と悩みが具体的で、入口の一文が読者の実感（自己嫌悪）から始まっている。アクションが「15分・1行だけ」と最小単位まで具体化され、読後に自分で効果を確かめる導線がある。
