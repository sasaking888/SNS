# テーマ試着用サンプル投稿

> 3つのテーマ候補それぞれで「実際に投稿するならこんな感じ」を3本ずつ用意しました。
> 読み比べて「自分が書いてて楽しそう・続けられそう」と感じるものを選んでください。
> ※そのまま使ってもOKですが、投稿前に自分の言葉に直すのを推奨します。

---

## 案A: AI活用×副業の実践記録

### A-1(取り組み紹介型)
```
副業を始めるにあたって、AIに「仮想の会社」を作ってもらった。

秘書室・SNS部門・note部門・YouTube部門。
社員は全員AI、人件費ゼロ。

指示を出すと各部門が企画や下書きを作ってくれる。
ひとり社長の時代、ほんとに来てるかも。
```

### A-2(気づき共有型)
```
AIで副業を効率化して気づいたこと。

AIが速いのは「作業」であって「決断」じゃない。

投稿を100本下書きしてもらうのは一瞬。
でも「何を発信するか」を決めるのは結局自分。

AI時代に価値が残るのは、決める力かもしれない。
```

### A-3(失敗・リアル型)
```
正直に言うと、AI副業1日目の成果はゼロ円です。

やったこと:
・AIで仮想組織を構築
・発信テーマの検討
・投稿の下書きストック作り

収益が出るまでの過程、全部記録して公開していきます。
0→1の瞬間を一緒に見届けてください。
```

## 案B: 本業スキルの発信
> ※本業の分野が未確定のため、汎用フォーマットで作成。分野が決まったら差し替えます。

### B-1(あるある否定型)
```
「○○(本業分野)は△△が大事」ってよく言われるけど、
現場に□年いる立場から言うと、実は違う。

本当に差がつくのは──(本業での気づきを入れる)

教科書には載ってない、現場だけの感覚です。
```

### B-2(初心者向け解説型)
```
○○を始めたい人がまずやるべきこと3つ。

1. (基本の一歩)
2. (見落としがちな準備)
3. (現場で学んだコツ)

□年やってきて、結局これが一番の近道でした。
```

### B-3(経験談型)
```
新人の頃、○○で大失敗した話。

(失敗エピソード)

あの時学んだのは「(教訓)」ということ。
同じ道を通る人の参考になれば。
```

## 案C: 趣味・得意分野の発信
> ※ジャンル未確定のため、汎用フォーマットで作成。

### C-1(熱量共有型)
```
○○(趣味)歴□年の自分が、
何度も戻ってくる「これだけは外せない」もの3選。

1. (定番だけど理由がある選択)
2. (意外と知られていない良品)
3. (自分だけのこだわり)

異論は認めます。むしろ聞かせてください。
```

### C-2(初心者の壁型)
```
○○を始めて最初に挫折しかけたポイントと、
乗り越えた方法。

挫折ポイント: (あるある)
突破法: (自分なりの工夫)

これから始める人、ここだけは知っておいて損ないです。
```

### C-3(比較レビュー型)
```
○○の「AとB、結局どっち?」問題。

□年使い比べた結論:
・Aが向いてる人 → (特徴)
・Bが向いてる人 → (特徴)

自分は(選択)派です。理由は(一言)。
```

## 読み比べのポイント

- **書き足したくなったのはどれ?** → そのテーマが一番熱量を持てる証拠
- **穴埋め(○○)がすぐ埋まったのはどれ?** → ネタの引き出しが多い証拠
- **1年続けるイメージが湧くのはどれ?** → 継続性が最重要

決まったら秘書までご連絡ください。選ばれたテーマで2週間分の投稿ストックを一気に作ります。
