# 発信テーマ選定の検討資料

> 作成: 2026-07-06 / 秘書室
> 状況: テーマは考え中。注力先は「一番コスパの良いもの」を希望 → X(SNS部門)を第一候補として推奨。

## なぜXから始めるのがコスパ最強か

| 観点 | X | note | YouTube |
|------|---|------|---------|
| 1コンテンツの制作時間 | 数分〜15分 | 1〜3時間 | 5〜20時間 |
| 反応が返ってくる速さ | 数時間 | 数日 | 数週間〜 |
| 失敗のコスト | ほぼゼロ | 低 | 高 |
| 収益化までの距離 | 間接(導線役) | 近い(有料記事) | 遠いが最大 |

**戦略**: X でフォロワーと「ウケるネタの感覚」を育てる → 反応が良かったネタを note 記事に膨らませて収益化 → 軌道に乗ったら YouTube へ展開。3部門は独立ではなく「X = 実験場、note = 収益化、YouTube = 拡大」のパイプラインとして使う。

## テーマ候補3案の比較

### 案A: AI活用×副業の実践記録
- **内容**: Claude Code等のAIを使って副業を立ち上げる過程をそのまま発信
- **強み**: 今まさにやっていることなのでネタが自動的に生まれる。市場の関心が高い。「仮想組織で副業運営」という切り口は差別化になる
- **弱み**: 発信者が多く競争は激しい。実績が出る前は説得力を作る工夫が必要
- **収益化ルート**: note有料記事(セットアップ手順・プロンプト集) → 教材・テンプレート販売

### 案B: 本業スキルの発信
- **内容**: 本業で持っている専門知識をコンテンツ化
- **強み**: 権威性・独自性を出しやすい。競合が相対的に少ない領域を狙える
- **弱み**: 本業との兼ね合い(会社規定・身バレ)の確認が必要
- **収益化ルート**: note有料記事 → 相談・コンサル → 教材

### 案C: 趣味・得意分野の発信
- **内容**: 好きなこと・詳しいジャンルを発信
- **強み**: 継続が最も苦にならない。熱量が文章に乗る
- **弱み**: ジャンル次第で収益化の距離が変わる。「好き」と「稼げる」の交点を探す必要あり
- **収益化ルート**: ジャンル依存(アフィリエイト、有料記事、案件など)

## 判断のための3つの質問(オーナー宛)

1. **ネタ切れせず1年続けられそうなのはどれ?** (継続性が最重要。フォロワーは3〜6ヶ月の継続で伸び始める)
2. **本業の就業規定的に出せない情報はあるか?** (案Bの前提確認)
3. **サンプル投稿を読み比べて「自分が書いてて楽しそう」なのはどれ?** → `sns/posts/2026-07-06-theme-trial-drafts.md` に試着用サンプルを用意済み

## 秘書からの推し

**案Aを軸に、案B/Cの要素を混ぜる**のが現実的だと考えます。理由:
- ネタが日々の活動から自動供給される(ネタ探しの時間 = ゼロ)
- この仮想組織の運営記録そのものがコンテンツになる(一石二鳥)
- 実績が小さいうちは「過程の共有」が最も誠実で共感を呼びやすい
